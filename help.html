关于 TrashVPN

TrashVPN 是一个由个人开发者开发和维护的免费 VPN 软件。在开发过程中，开发者使用了 Codex 辅助编写、调整与检查部分代码和网页内容，但产品的运营、发布与维护仍由开发者负责。

目前 TrashVPN 支持 Windows 和 Android 客户端。你可以在下载页面获取现有版本，并查看对应客户端的隐私政策和开源许可证信息。

由于项目采用了成本极低的运行方式，服务器和日常维护的支出相对可控，因此当前服务免费提供使用。 但如果未来节点被 Cloudflare 或防火墙彻底掐断，可能会选择终止软件服务或推出付费节点，走一步看一步吧。

为什么可提供的国家这么少

TrashVPN 的连接会涉及 Cloudflare 网络。同一个节点在不同运营商、不同地区甚至不同时间的网络环境下， 可能会被 Cloudflare 路由到不同国家或地区，因此外层出口所显示的国家并不是固定不变的。

Cloudflare 会综合判断用户的物理距离、目标地区服务器的部署数量，以及各地区服务器当前的负载情况， 决定一次连接最终路由到哪里。也就是说，外层 IP 对应的国家通常无法由 TrashVPN 直接指定。 因此我没法增加节点的国家数量，这个不是我能改变的。

除了外层 IP，连接中还存在可配置的内层 IP，也就是 proxy IP。当前 proxy IP 会从香港、新加坡、日本、美国四个地区中随机分配给用户。访问采用 Cloudflare CDN 服务的网站时，流量通常会走 proxy IP；访问其它网站时，则通常体现为外层 IP 的路由结果。

因此，节点实际所在地可能会与软件内显示的国家不同，软件内显示的国家大家就图一乐吧。

会有限额吗

会。Cloudflare 存在账户级使用限额，限额会在每天北京时间早上 8 点重置。 当某个账户达到限额后，关联到该账户的节点可能暂时无法继续使用。

一个账户可能会同时分配给多位用户，因此出现限额并不一定代表是你个人造成的； 也可能是同一账户下其他用户共同产生的使用量累积结果。

为了降低突然不可用带来的影响，客户端会每小时获取一次账户的使用量信息。当账户接近限额时， 客户端会重新获取最新节点，尽可能切换到仍可使用的资源。

你也可以前往状态页面查看整体负载的大致分布。状态数据用于辅助判断服务可用性， 并不代表某一台设备的精确流量统计。

  • 不会以“开通服务”“解除限制”“续费”或“会员升级”等理由向你收取费用或要求转账。
  • 不会要求你提交银行卡信息、支付密码、身份证件、短信验证码、邮箱密码或账户恢复代码。
  • 不会要求你下载与 TrashVPN 无关的清理工具、破解工具、远程控制软件或来源不明的证书配置。
  • 不会私下提供所谓“特殊节点”并要求你绕过官网从不明地址下载文件。
如果某个页面或联系人提出以上要求，请停止操作，不要提供敏感信息或运行未知文件。
